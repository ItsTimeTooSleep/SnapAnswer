# SnapAnswer - AI Answer Assistant

一个浏览器插件，让用户可以选中文字并通过快捷键快速获取 AI 的简化回答，自动复制到剪切板。

## 功能特点

- 🚀 **快速回答**: 选中文字，按下快捷键即可获得 AI 回答
- 🎨 **精美界面**: 现代化的弹出界面设计
- ⌨️ **自定义快捷键**: 支持自定义快捷键组合
- 🌍 **智能语言检测**: 自动识别文本语言并用相同语言回答
- 📋 **自动复制**: 回答自动复制到剪切板
- 🔒 **隐私保护**: API Key 仅本地存储，不会上传到服务器

## 安装步骤

1. **下载扩展文件**
   - 下载整个项目文件夹

2. **加载扩展**
   - 打开 Chrome 浏览器
   - 访问 `chrome://extensions/`
   - 开启右上角的"开发者模式"
   - 点击"加载已解压的扩展程序"
   - 选择项目文件夹

3. **配置扩展**
   - 点击浏览器工具栏上的 SnapAnswer 图标
   - 设置您的 DeepSeek API Key
   - 自定义快捷键（默认：Ctrl+Shift+Q）

## 使用方法

1. **选择文本**: 在网页上选中您想要询问的文字
2. **按下快捷键**: 使用您设置的快捷键（默认 Ctrl+Shift+Q）
3. **获取回答**: AI 回答将自动复制到剪切板
4. **粘贴使用**: 在需要的地方粘贴回答

## 设置说明

### 快捷键设置
- 支持 Ctrl/Cmd + Shift + 字母/数字组合
- 点击快捷键输入框并按下您想要的组合键
- 建议使用不与其他应用冲突的组合

### Key 获取
- 询问作者获取

## 技术特性

- **Manifest V3**: 使用最新的 Chrome 扩展标准
- **异步处理**: 非阻塞的 API 调用和用户界面
- **错误处理**: 完善的错误提示和重试机制
- **响应式设计**: 适配不同屏幕尺寸

## 故障排除

### 无法获取回答
- 检查网络连接
- 验证 API Key 是否正确
- 确认是否选中了有效文本

### 快捷键无效
- 检查是否与其他应用冲突
- 重新设置快捷键
- 刷新页面后重试

### API 调用失败
- 检查 API Key 是否有效
- 确认 API 配额是否充足
- 稍后再试（可能是请求频率限制）

## 隐私说明

- 🔒 API Key 仅存储在本地浏览器中
- 📊 不会收集或上传任何用户数据
- 🌐 仅在用户主动请求时调用 API
- 🗑️ 不保存任何查询历史

## 开发信息

- **框架**: Vanilla JavaScript
- **样式**: CSS3 + 渐变设计
- **图标**: SVG 矢量图形
- **API**: DeepSeek Chat API
